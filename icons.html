Icons documentation

Font Awesome

Cascade Framework contains the Font Awesome iconic webfont (v3.2.1), which includes a total of 361 icons. This font is licensed under the SIL Open Font License, is a free and open source license specifically designed for fonts and related software provides a legal framework and infrastructure for worldwide development, sharing and improvement of fonts and related software in a collaborative manner. It enables font authors to release their work under a common license that allows bundling, modification and redistribution.

Basicly, this means you can use this font in any project you like, just like the rest of Cascade Framework. Below, you get an overview of the different icons included in this font.
